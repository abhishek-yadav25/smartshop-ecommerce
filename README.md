# SmartShop – Django eCommerce Website

SmartShop is a full-stack eCommerce web application inspired by Amazon.  
It allows users to browse products, search items, add them to a cart, and complete purchases using online payment or cash on delivery.

## Features

- User Registration and Login
- Product Listing
- Product Search
- Product Detail Page
- Add to Cart
- Update Cart Quantity
- Remove Items from Cart
- Checkout System
- Razorpay Payment Gateway Integration
- Cash on Delivery Option
- Email Order Confirmation
- Responsive UI using Bootstrap

## Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- Bootstrap
- Razorpay Payment API
